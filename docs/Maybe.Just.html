andThen(transform) → {Maybe.<B>}

Source:
  • maybe/Maybe.js, line 528
Since:
  • 1.0.0-beta
Inherited From:
  • Maybe#andThen
See:
  • Maybe#chain

Apply a transformation to the Maybe if it is an instance of "Just". Otherwise, ignore the transformation and return the instance. This is how you can switch from a 'Just' to 'Nothing' instance and stop subsequent transformations from being applied. An alias for Maybe#chain

Example
// andThen Maybe a :: (a -> Maybe b) -> Maybe b
import { Maybe } from 'zoomjs';

const valid = Maybe.Just.of('yay!');
const invalid = Maybe.Nothing;

const toUpper = x => Maybe.Just.of(x.toUpperCase());

valid.andThen(toUpper); // Just('YAY!')
invalid.andThen(toUpper); // Nothing
