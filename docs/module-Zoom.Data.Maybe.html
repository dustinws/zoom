(static) ap(left, right) → {Maybe}

Source:
  • data/Maybe.js, line 137
Since:
  • v1.15.0
Apply a transformation to the Maybe if it is an instance of "Just". Otherwise, ignore the transformation and return the instance.
Example
import { Maybe } from '@dustinws/zoom/packages/data';

const valid = Maybe.Just.of('yay!');
const failure = Maybe.Nothing;

const toUpper = Maybe.of(x => x.toUpperCase());

Maybe.ap(toUpper, valid); // Just('YAY!')
Maybe.ap(toUpper, invalid); // Nothing

(static) chain(transform, maybe) → {Maybe}

Source:
  • data/Maybe.js, line 84
Since:
  • v1.15.0
Apply a transformation to the Maybe if it is an instance of "Just". Otherwise, ignore the transformation and return the instance. This is how you can switch from a 'Just' to 'Nothing' instance and stop subsequent transformations from being applied.
Example
import { Maybe } from '@dustinws/zoom/packages/data';

const valid = Maybe.Just.of('yay!');
const invalid = Maybe.Nothing;

const toUpper = x => Maybe.Just.of(x.toUpperCase());

Maybe.chain(toUpper, valid); // Just('YAY!')
Maybe.chain(toUpper, invalid); // Nothing
