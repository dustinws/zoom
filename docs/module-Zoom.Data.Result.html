(static) ap(left, right) → {Result}

Source:
  • data/Result.js, line 136
Since:
  • v1.15.0
Apply a transformation to the Result if it is an instance of "Ok". Otherwise, ignore the transformation and return the instance.
Example
import { Result } from '@dustinws/zoom/packages/data';

const valid = Result.Ok.of('yay!');
const failure = Result.Err.of('nay!');

const toUpper = Result.of(x => x.toUpperCase());

Result.ap(toUpper, valid); // Ok('YAY!');
Result.ap(toUpper, invalid); // Err('nay!');

(static) chain(transform, result) → {Result}

Source:
  • data/Result.js, line 83
Since:
  • v1.15.0
Apply a transformation to the Result if it is an instance of "Ok". Otherwise, ignore the transformation and return the instance. This is how you can switch from a 'Ok' to 'Err' instance and stop subsequent transformations from being applied.
Example
import { Result } from '@dustinws/zoom/packages/data';

const valid = Result.Ok.of('yay!');
const invalid = Result.Err.of('nay!');

const toUpper = x => Result.Ok.of(x.toUpperCase());

Result.chain(toUpper, valid); // Ok('YAY!');
Result.chain(toUpper, invalid); // Err('nay!');

result Result The result
