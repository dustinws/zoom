<static> guard(Input, The, args)

Guard a function's input / output types. Acts as a no-op, but will log the TypeError to the console in development.
Parameters:
Name Type Argument Description
Input Array.<function()> / output contracts
The function function to validate
args Any <repeatable>
Since:
  • v1.14.0
Returns:
Type
Validation
Example
import { guard, number } from '@dustinws/zoom/packages/check';

const add = guard([number, number, number], (a, b) => a + b);

add(1, 3) // 4
add(1, '3') // 13 (logs TypeError outside of production)
